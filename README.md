### 01_Nonlinear_Equations_Root_Finding

#### 📌 Тема: Чисельні методи розв'язання нелінійних рівнянь

**Мета:** знаходження коренів рівняння виду `f(x) = 0` із заданою точністю `ε`.

#### Реалізовані методи

| Метод | Опис |
|-------|------|
| **Метод половинного ділення (дихотомії)** | Ґрунтується на послідовному звуженні інтервалу `[a, b]`, на якому функція змінює знак. На кожному кроці інтервал ділиться навпіл і обирається та половина, де знак функції змінюється. |
| **Метод хорд** | Використовує лінійну інтерполяцію між кінцями інтервалу для наближення до кореня. Збігається швидше за дихотомію, але чутливіший до вибору початкового інтервалу. |

#### Рівняння

Тестове рівняння: `x - sin(x) = 0.25` → `f(x) = x - sin(x) - 0.25`

#### Версії

- **`lab1_bevziuk_1.py`** — консольна версія. Виводить ітерації, корінь та значення функції у термінал. Ідеально для розуміння алгоритму.
- **`lab1_bevziuk_2.py`** — повноцінний GUI з:
  - введенням довільної функції `f(x)`
  - побудовою графіка функції
  - графіком збіжності обох методів
  - таблицею ітерацій
  - автоматичним округленням результату до точності `ε`

#### Порівняння методів

| Критерій | Дихотомія | Хорди |
|----------|-----------|-------|
| Збіжність | Гарантована | Залежить від інтервалу |
| Швидкість | Лінійна | Надлінійна |
| Кількість ітерацій | Більша | Менша |

### 02_Linear_Equations_Systems_Gauss_Seidel

#### 📌 Тема: Розв'язання систем лінійних рівнянь методом Зейделя

**Мета:** розв'язання системи `A·x = b` ітераційним методом Зейделя з перевіркою умови збіжності.

#### Метод

**Метод Зейделя (Gauss–Seidel)** — ітераційний метод, у якому на кожному кроці нові значення невідомих обчислюються послідовно з використанням уже оновлених значень поточної ітерації.

Формула для `i`-го компонента:

```
x_i^(k+1) = (b_i - Σ_{j<i} a_ij·x_j^(k+1) - Σ_{j>i} a_ij·x_j^(k)) / a_ii
```

#### Умова збіжності

Перевіряється **діагональне переваження** матриці `A`:

```
|a_ii| > Σ_{j≠i} |a_ij|   для кожного рядка i
```

Якщо умова не виконується — програма попереджає про можливу розбіжність.

#### Версії

- **`lab2_bevziuk_1.py`** — один графік траєкторії `(x₁, x₂)` на Canvas.
- **`lab2_bevziuk_2.py`** — три графіки траєкторій: `(x₁, x₂)`, `(x₁, x₃)`, `(x₂, x₃)`, що дає повнішу картину збіжності.

#### Можливості GUI

- Введення матриці `3×3` та вектора `b`
- Задання точності `ε`
- Візуалізація траєкторії збіжності (червоні точки — проміжні, зелена — результат)
- Таблиця всіх ітерацій

### 03_Nonlinear_Equations_Systems_Simple_Iteration

#### 📌 Тема: Розв'язання систем нелінійних рівнянь методом простої ітерації

**Мета:** знаходження розв'язку системи нелінійних рівнянь виду:

```
x = φ₁(x, y)
y = φ₂(x, y)
```

#### Метод

**Метод простої ітерації:**

```
x_{k+1} = φ₁(x_k, y_k)
y_{k+1} = φ₂(x_k, y_k)
```

#### Умова збіжності

Перевіряється величина:

```
q = max(|∂φ₁/∂x| + |∂φ₁/∂y|, |∂φ₂/∂x| + |∂φ₂/∂y|)
```

Якщо `q < 1` — метод збігається. Програма обчислює `q` і повідомляє користувача.

#### Тестова система

```
φ₁(x, y) = e^x · cos(y) - 1
φ₂(x, y) = e^x · sin(y) + 1
```

#### Можливості GUI

- Введення початкової точки `(x₀, y₀)`
- Задання точності `ε`
- Перевірка умови збіжності перед обчисленням
- Графік траєкторії на площині `(x, y)`
- Таблиця ітерацій з номерами кроків

### 04_Function_Interpolation

#### 📌 Тема: Інтерполювання функцій

**Мета:** наближене обчислення значення функції в заданій точці за допомогою різних видів інтерполяції.

#### Функція

```
y = 4 - cos(x²)
```

#### Реалізовані методи

| Метод | Опис |
|-------|------|
| **Локальна лінійна інтерполяція** | Будується лінійний поліном Лагранжа на двох найближчих вузлах до точки `x*` |
| **Локальна квадратична інтерполяція** | Будується квадратичний поліном Лагранжа на трьох найближчих вузлах |
| **Глобальна інтерполяція Лагранжа** | Використовує всі задані вузли для побудови єдиного полінома |

#### Формула полінома Лагранжа

```
L(x) = Σ y_i · Π (x - x_j) / (x_i - x_j),   i ≠ j
```

#### Відносна похибка

```
δ = |y_interp - y_exact| / |y_exact| · 100%
```

#### Можливості GUI

- Введення 6 вузлів інтерполяції (за замовчуванням: `-1.0, 0.0, 0.8, 1.355, 1.773, 2.0`)
- Задання точки `x*` для обчислення
- Таблиця вузлів із значеннями `y`
- Таблиця результатів: значення кожного методу та відносна похибка
- **Три графіки** з порівнянням:
  - точна функція vs локальна лінійна інтерполяція
  - точна функція vs локальна квадратична інтерполяція
  - точна функція vs глобальна інтерполяція Лагранжа

### 05_Numerical_Integration

#### 📌 Тема: Чисельне обчислення визначеного інтеграла

**Мета:** наближене обчислення інтеграла:

```
I = ∫[a,b] f(x) dx
```

#### Підінтегральна функція

```
f(x) = 1 / √(2x² + 1)
```

**Межі інтегрування:** `a = 0.8`, `b = 1.6`

#### Реалізовані методи

| Метод | Опис |
|-------|------|
| **Метод лівих прямокутників** | Інтеграл наближується сумою площ прямокутників з висотою у лівому кінці кожного підінтервалу. Крок `h` задається користувачем. |
| **Метод трапецій з подвійними перерахунками** | Автоматичне подвоєння кількості підінтервалів доки оцінка похибки `|I₂ - I₁| / 3` не стане меншою за `ε`. |

#### Формули

**Ліві прямокутники:**

```
I ≈ h · Σ f(a + i·h),   i = 0, ..., n-1
```

**Трапеції:**

```
I ≈ h · [f(a)/2 + Σ f(a + i·h) + f(b)/2],   i = 1, ..., n-1
```

#### Можливості GUI

- Задання кроку `h` для методу прямокутників
- Задання точності `ε` для методу трапецій
- Вивід кількості розбиттів, значення інтеграла та оцінки похибки

### 06_Euler_Method_ODE

#### 📌 Тема: Розв'язання задачі Коші методом Ейлера

**Мета:** чисельне розв'язання звичайного диференціального рівняння першого порядку.

#### Задача Коші

```
y' = 3x² / (x³ + y + 1)
y(1) = 0
```

**Інтервал:** `[1, 2]`

#### Метод Ейлера

Найпростіший один кроковий метод:

```
y_{k+1} = y_k + h · f(x_k, y_k)
x_{k+1} = x_k + h
```

де крок `h` обчислюється автоматично:

```
h = (b - x₀) / n,   n = ⌈(b - x₀) / ε⌉
```

#### Можливості GUI

- Задання початку `x₀` (за замовчуванням `1`)
- Задання кінця інтервалу `b` (за замовчуванням `2`)
- Задання точності `ε` (визначає крок `h`)
- **Графік** наближеного розв'язку з інтерактивними інструментами (масштабування, переміщення)
- **Таблиця ітерацій:** номер кроку `k`, `x_k`, `y_k`, `f(x_k, y_k)`
- Вивід підсумкової інформації: крок `h`, кількість кроків, останнє наближення
